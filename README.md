# Smart City Project

A web-based Smart City application built with Spring Boot and MySQL.

## Features
- User Registration and Login
- Admin Dashboard
- News Articles
- Business Directory
- Forum Posts
- Market Rates
- Subscriptions

## Technologies Used
- Java 17
- Spring Boot 4.0.6
- MySQL 8.0
- Hibernate / JPA
- Thymeleaf
- Maven

## Setup Instructions
1. Clone the repository
2. Create MySQL database: `CREATE DATABASE smartcitydb;`
3. Update `application.properties` with your MySQL username and password
4. Run the project
5. Open `http://localhost:8080` in your browser
